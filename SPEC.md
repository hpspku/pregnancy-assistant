# 怀孕助手 App - 产品规范

## 1. 概念与愿景

一款温暖、贴心的孕期助手应用，帮助准妈妈们追踪孕期进度、了解胎儿发育、提供健康建议。以小红书为设计灵感，打造清新、温暖、有爱的视觉体验。

## 2. 设计语言

### 美学方向
- 参考小红书的温暖清新风格，融合母婴用品的柔和质感
- 圆润的卡片设计，柔和的阴影，营造安全感

### 色彩系统
- Primary: #FF6B9D (玫瑰粉)
- Secondary: #FFB6C1 (浅粉)
- Accent: #FFA07A (珊瑚橙 - 孕期阶段标识)
- Background: #FFF5F7 (淡粉白)
- Card: #FFFFFF
- Text Primary: #2D3436
- Text Secondary: #636E72
- Success: #00B894 (产检绿灯)
- Warning: #FDCB6E (提醒)

### 字体
- 主字体: "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei", sans-serif
- 标题: 600 weight
- 正文: 400 weight

### 间距系统
- 基础单位: 8px
- 卡片间距: 16px
- 内边距: 20px
- 圆角: 16px (卡片), 24px (大卡片), 50% (头像)

### 动效哲学
- 页面切换: slide + fade, 300ms ease-out
- 卡片进入: fadeInUp, 400ms, staggered 80ms
- 按钮交互: scale(0.96) on press, 150ms
- 加载状态: 优雅的心跳动画

## 3. 布局与结构

### 页面结构
1. **首页** - 孕期倒计时 + 当前状态卡片 + 每日建议
2. **孕期追踪** - 按周展示胎儿发育图文
3. **待办清单** - 孕期各阶段准备事项
4. **健康记录** - 体重、血压、产检记录
5. **小贴士** - 每日孕育知识

### 导航
- 底部 Tab 栏，5个主要功能入口
- 当前选中项高亮 + 轻微放大

## 4. 功能与交互

### 核心功能

#### 4.1 孕期计算器
- 输入末次月经日期
- 自动计算：孕周、预产期、剩余天数
- 进度条可视化展示孕期进度

#### 4.2 胎儿发育追踪
- 每周展示：胎儿大小比喻（如：柠檬→芒果→西瓜）
- 胎儿身长、体重数据
- 本周发育特点（3-5条）
- 准妈妈身体变化
- 注意事项

#### 4.3 待办清单
- 按孕期阶段分类（早/中/晚期）
- 待办事项勾选
- 完成进度统计
- 添加自定义事项

#### 4.4 健康记录
- 体重记录（曲线图）
- 血压记录
- 产检提醒
- 重要产检时间节点提示

#### 4.5 每日知识
- 每日一条孕育小贴士
- 饮食建议、运动建议、情绪调节
- 可收藏到"我的收藏"

### 交互细节
- 卡片点击：轻微下沉 + 阴影变化
- 切换周数：平滑滑动
- 完成任务：打勾动画 + 撒花效果
- 页面切换：底部导航高亮 + 内容区切换

## 5. 组件清单

### 5.1 孕期进度环
- 圆形进度条，外圈显示百分比
- 中心显示当前孕周
- 颜色随进度渐变

### 5.2 胎儿大小卡片
- 水果类比图标（大emoji）
- 尺寸、体重数据
- 发育状态标签

### 5.3 待办事项卡片
- 圆形复选框
- 事项标题 + 描述
- 孕期阶段标签（早期/中期/晚期）
- 完成态：文字划线 + 透明度降低

### 5.4 健康数据卡片
- 数据数值（大字）
- 单位（小字）
- 趋势箭头（上升/下降/平稳）
- 记录时间

### 5.5 Tab 导航
- 5个图标 + 文字
- 选中态：图标变色 + 文字加粗
- 轻微的弹跳动画

## 6. 技术方案

### 技术栈
- 纯 HTML5 + CSS3 + JavaScript
- 无框架依赖，便于 GitHub Pages 部署
- LocalStorage 数据持久化

### 数据结构
```javascript
// 用户数据
{
  lastPeriodDate: "2025-01-15", // 末次月经
```
